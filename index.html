AuroPay

Now known as TalamPay, AuroPay is a mobile application that acts as an online payment system for a village in India called Auroville. This village has an isolated payment system that doesn't rely on any known banks or online payment systems, previously they used the traditional paper payment system where people had to sign a paper to process any transaction.

During the summer of 2020, I alongside a senior developer, created a React Native application that allowed people in Auroville to send money to each other through their devices, it also allowed QR code scanning to allow users to send quick payments.

  • Developed using React Native, it runs on both iOS and Android Devices
  • While the name has changed, the application is currently being used by 100 buisness that provide goods and services in the village
  • Leard the HTTP protocal and making fetch request to web applications
  • Used HTTP fetch request to the online banking web application, with that I was able to process transactions to essentially create the mobile friendly payment system
  • Senior developer appreacited my appitude to learning skills to make the required features
  • Project was a part of a volunteer position during the summer of 2020
